СИНТЕЗ
Это мастер-страница: она не вводит новых концептов, а собирает все разделы базы в один каркас и показывает на кейсах, как их комбинировать. Детали каждого слоя — в его разделе; здесь — как они складываются в сделку.
01 Зачем синтез why
L1 · База

Ни один слой не торгуется в одиночку. Каждый отвечает на свой вопрос, а сила — в стеке, где несколько независимых слоёв сходятся в одну историю.

Главное правилоОдин слой = гипотеза; стек слоёв у заранее выбранной зоны = сделка. Порядок чтения: контекст → локация → поток → триггер → риск.
L2 · Чтение
  • Слои не равны: контекст (AMT/режим) и локация (VP/levels) — первичны
  • Поток (CVD/footprint) — подтверждение; топливо (OI/funding/liq) — качество движения; риск — выживание
  • 90% «сигналов» — шум, пока их не подтвердил контекст уровня и поток
L3 · Эдж
ЭджНе ищи «грааль-индикатор». Ищи совпадение независимых слоёв у заранее найденной зоны. Чем больше слоёв в одну сторону — тем выше вероятность; конфликт слоёв — повод уменьшить размер или пропустить.
02 Карта слоёв: какой вопрос решает каждый map
L1 · База
ВопросСлой / разделы
Какой режим?AMT, OF Regime
Где? (стоимость, уровни)Volume Profile, TPO, Key levels, VWAP
Кто давит? (агрессия)CVD, Footprint, Imbalances, Volume
Топливо / перегрев?Open Interest, Funding, Liquidations
Ликвидность: стены, стопы?Orderbook heatmap, HL SL/TP
Сколько? (выживание)Математика и риск
L2 · Чтение

Читай сверху вниз: режим → где → кто → топливо → риск. Пропуск верхних слоёв (вошёл по «сигналу» без контекста и локации) = торговля шума.

L3 · Эдж
Дубли и конфликтыСлои-дубли (spot CVD ↑ + footprint absorption + OI build) усиливают одну историю. Слои-конфликты (perp ведёт, а спот против) — это предупреждение о хрупкости (кейс C).
03 Каркас сделки: 5 шагов framework
L1 · База

Универсальный пайплайн — он одинаков для любого сетапа:

  • 1 · Контекст — режим (AMT), bias (VWAP / open), где в value
  • 2 · Локация — A-зона: key level + VP-наки + стена / стопы сошлись
  • 3 · Поток — у зоны: absorption / imbalance + CVD + OI / funding / liq
  • 4 · Триггер — accept / reject / reclaim / sweep+reclaim + footprint
  • 5 · Риск — стоп за структуру, R до следующей зоны, размер ≤ 1–2%
L2 · Чтение
ГейтВход появляется, только когда совпали все 5. Нет локации — нет сделки; нет потока — это гипотеза; нет посчитанного риска — это азарт.
L3 · Эдж
СвязьЭто та же логика «zone → regime → trigger → invalidation → magnet» из OF Regime, расширенная на VP / levels (локация) и риск.
04 Кейс A: разворот на уровне (absorption) reversal
L1 · База
  • Локация: цена долетела до composite naked POC / стека поддержки (Key levels + VP)
  • Топливо: liq-flush — каскад long-ликвидаций + OI flush (капитуляция)
  • Поток: footprint absorption (forced sells впитаны) + spot-CVD дивергенция (price LL, spot CVD HL)
  • Триггер: reclaim уровня → лонг; стоп за sweep-лой, R до vPOC выше
L2 · Чтение

Каждый слой по отдельности — гипотеза. Вместе (уровень + flush + absorption + дивергенция) = высоковероятный разворот: продавцов вымыли, а пассив их впитал.

L3 · Эдж
ЛовушкаБез reclaim не лови нож: liq-flush может продолжиться в следующий кластер. Absorption — гипотеза, пока цена не отбила вверх.
05 Кейс B: продолжение / пробой continuation
L1 · База
  • Контекст: баланс / inside day, выход вверх (AMT imbalance, LVN над VAH)
  • Локация: пробой VAH + LVN-пролёт к naked POC выше
  • Поток: OI растёт (новые лонги) + spot CVD ведёт (реальный спрос) + stacked buy imbalances в импульсе
  • Триггер: VWAP / VAH reclaim на ретесте → лонг продолжения; стоп под VAH, R до naked POC
L2 · Чтение

Spot-led + OI build + acceptance за границей = здоровый тренд, а не perp-сквиз. Ретест удержан → инициатива принята.

L3 · Эдж
РазвилкаЕсли spot НЕ ведёт (perp-only) и funding скачет — это уже не кейс B, а кейс C (фейд). Один и тот же рост, разный поток → противоположная сделка.
06 Кейс C: фейд ловушки (perp-led) fade
L1 · База
  • Локация: ралли в HTF resistance / OB ask-стену
  • Топливо: perp CVD ведёт, spot плоский / против; funding-спайк (лонги переполнены, платят); OI растёт (заёмный)
  • Поток: footprint absorption на топе (агрессивные покупки впитаны) + дельта-дивергенция
  • Триггер: failed reclaim / loss локального low → шорт-фейд; стоп за хай, R до VWAP / наки ниже
L2 · Чтение

Это «хрупкий рост» из OF Regime: вид бычий, но спот не верит, плечо переполнено → фейд у стены, когда инициатива выдыхается.

L3 · Эдж
ЛовушкаФейд — только по триггеру (reject / loss структуры), не по «funding высокий» вслепую. Перегрев тянется дольше, чем кажется.
07 Конфлюэнс-матрица confluence
L1 · База
Комбо слоёвСмысл
key level + naked POC + стена / стопыA-локация (где готовить сделку)
spot CVD ведёт + footprint absorption + OI buildподтверждённая инициатива → continuation
liq-flush + absorption + хвост у уровнякапитуляция → разворот (кейс A)
perp ведёт + funding-экстрим + OI build (без спота)хрупко → фейд (кейс C)
imbalance + acceptance + VWAP reclaimпробой → continuation (кейс B)
L2 · Чтение
  • Больше независимых слоёв в одну сторону = выше вероятность
  • Конфликт слоёв (спот против перпа) = меньше размер или пропуск
L3 · Эдж
Определение A-сделки≥ 3 независимых слоя в одну сторону у заранее выбранной зоны + чистый триггер + R ≥ 2. Меньше — это наблюдение, а не сделка.
08 Чек-лист и анти-чек-лист checklist
L1 · База
СутьПеред кликом пройди 5 шагов. Если на любом «нет ответа» — это наблюдение, а не сделка.
L2 · Чтение
✓ Pre-trade
① Режим/контекст определён? ② Цена у A-зоны (стек уровней)? ③ Поток подтверждает (CVD/footprint/OI)? ④ Чистый триггер (accept/reject/reclaim)? ⑤ Стоп / R / размер посчитаны?
✗ Анти-чек-лист
Торговать середину диапазона · гнаться за perp-сквизом · ловить нож в каскаде · входить на одном «красивом» слое · путать absorption и imbalance · двигать стоп против себя.
L3 · Эдж
Каркас (резюме)контекст (AMT/regime) → локация (VP/levels) → поток (CVD/footprint/OI/funding/liq) → триггер → риск (§18).
